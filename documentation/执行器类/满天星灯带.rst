满天星灯带
===================

.. figure:: /static/图片/乐动模块-满天星灯带.png
	:width: 65%
	:align: center


概述
--------------------
彩色满天星LED彩灯，高电平亮。



技术参数
-------------------

工作电压： VCC 3.3V -5V

长：2.0米

LED灯：5色，20颗


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND     负极 
VCC     正极  
=====  ======== 


示例程序
-------------------

**A按键按下，打开满天星灯带；B按键按下，则关闭。**
