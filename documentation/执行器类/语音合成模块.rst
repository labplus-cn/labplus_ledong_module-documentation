语音合成模块
===================

.. figure:: /static/图片/语音合成传感器.png
	:width: 55%
	:align: center

概述
--------------------
离线语音合成模块，支持英文、中文常见字符语音播报。



技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

通讯方式：串口

引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
TX     串口发送  
RX     串口接收
=====  ========
