概述
--------------------
MP3播放模块，可通过软串口发送指令操控播放u盘中保存的歌曲。


技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：32*48mm

固定安装：2个兼容乐高积木通孔

通讯方式：串口，波特率9600

文件格式：MP3/WAV



引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
TX     串口发送  
RX     串口接收
=====  ========
