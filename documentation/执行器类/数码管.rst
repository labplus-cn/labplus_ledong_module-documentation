数码管管模块
===================

.. figure:: /static/图片/数码管-H.png
	:width: 55%
	:align: center

概述
--------------------
4位7段LED数码管显示，可显示整形数字。



技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

显示范围：0-9999



引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
SCL	   I2C时钟 
SDA	   I2C数据
=====  ========
