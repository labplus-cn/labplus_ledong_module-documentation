激光发射
===================

	:width: 55%
	:align: center

概述
--------------------
内置激光发射头，高电平开启激光器，低电平熄灭。

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

控制方式：数字输入



引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
DI     数字输入 
NC     空
=====  ======== 

示例程序
-------------------

**A按键按下，发射激光；B按键按下，激光关闭。**
