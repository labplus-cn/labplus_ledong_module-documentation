风扇
===================

.. figure:: /static/图片/电机风扇.png
	:width: 55%
	:align: center

概述
--------------------
300直流电机，结合风扇叶，组合成风扇。



技术参数
-------------------

工作电压： VCC 3.3V -5V

转速：15000rpm

轴径：2mm



示例程序
-------------------

**A按键按下，打开电机风扇；B按键按下，则关闭。**
