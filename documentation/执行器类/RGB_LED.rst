RGB_LED
===================

.. figure:: /static/图片/RGB_LED.png
	:width: 55%
	:align: center

概述
--------------------
可实现256级亮度显示，16777216种颜色的全真色彩显示颜色。



技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

控制方式：RGB分量


引脚定义
-------------------

=====  =========== 
引脚    定义   
=====  ===========  
GND    地  
VCC    电源  
DI     控制信号输入
NC     空
=====  ===========
