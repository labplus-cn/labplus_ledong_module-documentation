超声波
===================

.. figure:: /static/图片/超声波-H.png
	:width: 55%
	:align: center

概述
--------------------
用于超声波定位、测距、避障等应用场景，采用I2C通讯将测距值返回主控。

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：32*48mm

固定安装：2个兼容乐高积木通孔

测量范围：5-250cm

引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
SCL     I2C时钟  
SDA     I2C数据
=====  ========
