PM2.5(新版，模块+探头)
====================================



概述
--------------------
PM2.5灰尘传感器在检测非常细的颗粒，用于进行PM2.5检测，可评估空气质量。



技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容塑料积木通孔

通讯方式：串口，波特率9600

输出检测值：0-2500μg/m³

上电稳定时间：≤10 s




引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
TX     串口发送  
RX     串口接收
=====  ========
