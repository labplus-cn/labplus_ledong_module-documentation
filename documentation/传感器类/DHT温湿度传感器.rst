概述
--------------------
DHT11是一款常用的温湿度传感器，具有较高的性价比和抗干扰能力


技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

工作电流：0.3mA（测量），60uA（待机）

温度范围：-20°C至60°C，精度为±1°C

湿度范围：5%至95%，精度为±5%RH。 


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
DI     数字信号输入/输出端 
NC     空
=====  ========
