摇杆
===================

.. figure:: /static/图片/摇杆-H.png
	:width: 55%
	:align: center

概述
--------------------
具有(X,Y)2轴模拟输出，可以制作遥控器用于控制运动物体的方向、速度等互动作品。


技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：模拟采样值

探测距离：0-5cm



引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
A1	   X轴模拟采样值 
A2	   Y轴模拟采样值
=====  ========
