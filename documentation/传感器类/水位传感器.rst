水位传感器
===================

.. figure:: /static/图片/水位.png
	:width: 55%
	:align: center

概述
--------------------
可用于探测水位高低，当水位增高时，传感器输出值将增大，反之将减小。


技术参数
-------------------

工作电压： VCC 3.3V -5V

固定安装：2个螺丝孔

数据类型：模拟采样值


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
AO	   模拟采样值  
NC     空
=====  ======== 


示例程序
-------------------

**掌控板OLED屏显示水位测量值，同时当测量值小于1000时，掌控板RGB灯亮。**
