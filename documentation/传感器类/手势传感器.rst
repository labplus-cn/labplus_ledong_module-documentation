手势传感器
===================

.. figure:: /static/图片/手势感应-H.png
	:width: 55%
	:align: center

概述
--------------------
用于检测手势变化（向上、向下、向左、向右、远近）

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：手势数据字符串



引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
SCL	   I2C时钟
SDA	   I2C数据
=====  ========
