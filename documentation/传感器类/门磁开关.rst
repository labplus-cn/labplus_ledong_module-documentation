门磁开关
===================

.. figure:: /static/图片/磁控开关-H.png
	:width: 55%
	:align: center

概述
--------------------
当有磁铁靠近时,干簧管合拢,输出高电平,否则输出低电平。

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：触发--1 未触发--0


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
DO     数字输出  
NC     空
=====  ========
