概述
--------------------
PM2.5灰尘传感器在检测非常细的颗粒，用于进行PM2.5检测。


技术参数
-------------------

工作电压： VCC 3.3V -5V

通讯方式：串口，波特率9600

输出：PM2.5检测值0-500μg/m³

-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND     地
VCC     电源
AO      输入
PLS     脉冲
