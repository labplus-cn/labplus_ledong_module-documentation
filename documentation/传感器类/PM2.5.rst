PM2.5
===================

概述
--------------------
PM2.5灰尘传感器在检测非常细的颗粒，可评估空气质量。


技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

检测种类：PMO.3~PM10

检测范围：5~2500μg/m3(微克/立方米)

输出方式：模拟值



引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
AO     模拟输出
NC     空
=====  ========
