心率血氧模块
===================

.. figure:: /static/图片/心率血氧传感器.png
	:width: 55%
	:align: center

概述
--------------------
心率血氧模块适用于指尖手指测量，不适用与手腕手指测量。

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：心率、血氧值

心率精确度：+1-5bpm(静态)，+/-10bpm(动态)


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
SCL	   I2C时钟 
SDA	   I2C数据
=====  ========

-------------------
