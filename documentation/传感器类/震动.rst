震动
===================

.. figure:: /static/图片/震动传感器-H.png
	:width: 55%
	:align: center

概述
--------------------
用于检测物体震动，能感应到物体微弱震动后，触发后输出高电平。


技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：震动--1 静止--0


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
DO     数字输出  
NC     空
=====  ========
