RFID
===================

.. figure:: /static/图片/RFID-H.png
	:width: 55%
	:align: center

概述
--------------------
能有效识别射频卡ID并读写信息，可模拟实现电子钱包的充值和扣费功能。



技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：ID和电子钱包信息

探测距离：0.5-1cm



引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
SCL     I2C时钟  
SDA     I2C数据
=====  ======== 




示例程序
-------------------

**1、按A键获取射频卡序列号并初始化设置并RGB亮红灯，按B键对射频卡充值并亮绿灯。**

**2、掌控板OLED屏显示RFID序列号和当前余额。。**
