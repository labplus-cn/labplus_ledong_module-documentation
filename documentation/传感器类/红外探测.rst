红外探测
===================

.. figure:: /static/图片/红外探测.png
	:width: 55%
	:align: center

概述
--------------------
探测测距传感器，检测正前方是否有障碍物，可探测范围3~30cm。

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：模拟采样值

探测距离：0-5cm


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
AO     模拟采样值
NC     空
=====  ======== 


示例程序
-------------------

1.将障碍物置于红外发射管前方，当红外反射回去的强度达到设定的值，DO输出高电平，指示灯点亮。

2.可通过调节电位器，设定阈值。阈值越大，可探测距离越远。


**程序功能：当有人接近触发红外探测传感器时，RGB灯亮绿灯，否则亮红灯。**
