旋钮电位器
===================

.. figure:: /static/图片/旋钮电位器.png
	:width: 55%
	:align: center

概述
--------------------
通过旋扭电位器调节AO输出模拟量,可结合其他模块作为调节控制。



技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：模拟采样值


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
AO	   模拟采样值 
NC     空
=====  ========
