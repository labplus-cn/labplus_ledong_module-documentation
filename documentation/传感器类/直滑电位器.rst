直滑电位器
===================

概述
--------------------
直滑电位器模块可以通过滑动手柄来调节电阻值，输出模拟信号，适用于需要调节电阻的应用场景。


技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：模拟值


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
AO     模拟输出  
NC     空
=====  ========
