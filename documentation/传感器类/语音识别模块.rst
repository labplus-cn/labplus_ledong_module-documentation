语音识别模块
===================

.. figure:: /static/图片/语音识别传感器.png
	:width: 55%
	:align: center

概述
--------------------
语音识别模块可以识别多种语音指令，适用于语音控制的应用场景。


技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：语音指令id,整形


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
TX     串口发送  
RX     串口接收
=====  ========
