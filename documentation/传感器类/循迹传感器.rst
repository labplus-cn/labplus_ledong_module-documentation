循迹传感器
===================

.. figure:: /static/图片/循迹传感器.png
	:width: 55%
	:align: center

概述
--------------------
按压式的开关数字输入模块，按下为高电平输出，释放为低电平输出。

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：按下--1 松开--0

引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
DO     数字输出  
NC     空
=====  ========
