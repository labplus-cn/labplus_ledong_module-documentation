人体感应
===================

.. figure:: /static/图片/人体感应-H.png
	:width: 55%
	:align: center

概述
--------------------
基于热释电红外传感器，能检测人或动物身体发射的红外线而输出电信号。

技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

数据类型：触发--1 未触发--0


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
DO     数字输出  
NC     空
=====  ========
