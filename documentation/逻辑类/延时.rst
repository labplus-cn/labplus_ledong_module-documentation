延时模块
===================


.. figure:: /static/图片/乐动模块-延时.png
	:width: 55%
	:align: center

概述
--------------------
对输入信号做延迟处理，延迟时间0-7秒可调。



技术参数
-------------------

工作电压： VCC 3.3V -5V

模块尺寸：24*48mm

固定安装：2个兼容乐高积木通孔

延时时间：0-7s可调


引脚定义
-------------------

=====  ======== 
引脚    定义   
=====  ========  
GND    地  
VCC    电源  
DI     数字输入  
NC     空
=====  ========
