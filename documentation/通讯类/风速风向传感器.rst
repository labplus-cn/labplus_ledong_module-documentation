概述
--------------------
气象站专用传感器，测量风速和风向，485信号通讯。



技术参数
-------------------

工作电压：12V

波特率：4800bps

风速测量范围：0-35m/s（0-12级）

风向测量范围：八方位

动态响应时间：≤2秒

配件：12V-DC电源适配器、转接线等




引脚 定义 
-------------------

=======  ======== 
引脚       定义   
=======  ========  
GND       地
VCC       电源  
RX        串口接收
TX        串口发送
=======  ========
